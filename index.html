PROJETOS

Linketinder

Um projeto fullstack feito utilizando Typescript, Groovy, Gradle, PostgreSQL. É uma aplicação que tem como objetivo ser uma plataforma que conecta candidatos a vagas de empresas. Nele, empresas podem se cadastrar para cadastrar vagas e dar 'likes' em candidatos que as interessem. Já os candidatos podem dar likes nas vagas em que acham que se encaixa

Projeto TFC

Primeiro projeto fullstack feito na trybe Feito utilizando NodeJS, Typescript, Express, Mocha, Chai, Docker É um simulador de um campeonato de futebol, onde podemos inserir e alterar partidas e ver a classificação dos times na tabela. Todo backend e os endpoints consumidos pelo front foram de minha autoria, realizados no curso da Trybe

Automação OCR Space

Projeto desenvolvido utilizando C# com objetivo de realizar uma automação utilizando a API OCR Space para extrair dados de invoices. Nesse projeto, são feitas chamadas na API pública que retorna dados lidos. Através desse retorno, tratamos para extrair os dados necessários
